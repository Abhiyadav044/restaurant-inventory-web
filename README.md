# Restaurant Inventory Dashboard

A full-stack restaurant inventory management app built with React and FastAPI.

## Live Demo
- Frontend: https://restaurant-inventory-web-psi.vercel.app
- Backend API: https://restaurant-inventory-web.onrender.com
- API Docs: https://restaurant-inventory-web.onrender.com/docs

## Features
- User login
- Add, edit, and delete inventory items
- Search and filter inventory
- Low stock and expiry tracking
- Dashboard cards for stock status
- Charts for stock analytics
- Responsive dashboard UI

## Tech Stack
- Frontend: React, Vite, CSS, Recharts
- Backend: FastAPI, SQLAlchemy, SQLite
- Deployment: Vercel, Render
